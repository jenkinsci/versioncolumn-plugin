The goal is to help making sure JVM versions in use on Master and Agents are reasonably aligned to avoid ClassFormatError and other cool errors.

Description Examples

Agent runtime must be greater or equal than the Controller bytecode level (strongly recommended minimum)

  • an agent running Java 6 or less will be disconnected from 2.32.3 Controller

  • an agent running Java 6 will not be disconnected from a 1.609 Controller

  • an agent running Java 7 will be disconnected from a 2.54 Controller

Agent major.minor must be equal to Controller major.minor JVM version (paranoid version)

  • an agent running 1.7 or less will be disconnected from a Controller running 1.8.112

  • an agent running 1.8.66 will not be disconnected from a Controller running 1.8.112

Agent JVM version must be exactly the same as Controller JVM version (paranoid++ version)

  • an agent running 1.8.66 will be disconnected from a Controller running 1.8.112
